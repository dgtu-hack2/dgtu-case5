Просмотр состояния программ

Для просмотра состояния программ необходимо выбрать одну из трех кнопок “Хорошее состояние”,“Требует внимания”, “Плохое состояние”.

Лист с образовательными программами

Полный список образовательных программ доступен сразу же после входа на страницу. После выбора одной из кнопок состояний можно увидеть образовательные программы

Просмотр состояния определенной программы

Чтобы ознакомиться с информацией и получить рекомендации, необходимо нажать на название образовательной программы

Подробности об образовательной программе

Подробнее об образовательной программе можно узнать нажав на кнопку “Посмотреть программу”

Оценки студентов
Просмотр списка студентов

Для просмотра списка студентов необходимо перейти на страницу "Оценки", кнопка перехода располагается на навигационной панели в левом углу экрана.

Фильтр для направлений

Для выбора определенной образовательной программы необходимо в поле "Образовательная программа..." написать название образовательной программы.

Переход на страницу к опеределнному студенту 

Чтобы ознакомиться с информацией о модулях определенного студента необходимо на странице "Оценка" нажать на кнопку с иконкой галочки в столбце "Переход".

Модуль и задания

После перехода по кнопке "Переход" Вы можете ознакомиться с модулями и заданиями определенного студента.

Просмотр компетенций
Переход на страницу компетенций

Для осуществления перехода на страницу компетенций необходимо выбрать в левом углу "Компетенции".

Список компетенций

Для выбора определенной компетенции необходимо в поле "Компетенции..." написать название необходимой компетенции и выбрать нужную.

Общая информация о компетенциях

После перехода на страницу по кнопке, Вы можете ознакомиться с общей информацией о компетенции.

Индикатор

Ниже общей информации о компетенции, Вы можете ознакомиться с ее индикаторами.

Просмотр образовательных программ
Переход на страницу образовательных программ.

Переход на страницу выбора образовательных программ осуществляется по кнопке слева "Программы" или для определенной образовательной программы с помощтю кнопки на странице "Мониторинг"-"Программы"-"Выбор программы"-"Посмотреть программу".

Лист с образовательными программами

Для выбора определенной образовательной программы необходимо в поле "Образовательные программы..." написать название необходимой образовательной программы и выбрать нужную.

Просмотр состояния определенной программы

Общие сведения об определенной программе Вы можете получить перейдя на страницу "Программы".

Модули образовательных программ

Подробнее о модулях Вы можете узнать нажав на блоке "Модули". 

Задачи образовательных программ

Подробнее о задачах Вы можете узнать нажав на блок "Модули"-"Задачи". 

Анализ образовательных программ
Переход на страницу анализа образовательных программ

Для перехода на страницу анализа образовательных программ необходимо нажать на кнопку "Анализ" слева на навигационной панели.

Выбор образовательных программ

Просмотр полного списка образовательных программ можно осуществить после перехоода на страницу "Анализ" Для выбора образовательных программ необходимо нажать на кнопку с иконкой галочки в столбце "Переход к сравнению". Можно найти определенную образовательную программу с помощью фильтра выше таблицы.

Сравнение образовательных программ

Необходимо в поле "Введите ссылку" ввести ссылку на сравниваемый результат и нажать на кнопку "Результат".

Результат

В поле "Результат" выводится результат сравнения с образовательной программой выбранной ранее.
